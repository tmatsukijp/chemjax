atto_chemjaxpv
=================

This is a plugin for Atto editor in Moodle that allows it to show into
preview window. To install unpack to lib/editor/atto/plugins/chemjaxpv, 
and visit the admin notifications page to install.  Then add the button
on the Atto toolbar settings page (for example next to links like 'preview = chemjaxpv').
